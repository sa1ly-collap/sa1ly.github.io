# Стажер бизнес-аналитик

## О себе

Студентка 4 курса направления «Бизнес‑информатика». Раньше работала администратором компьютерного клуба, поэтому хорошо понимаю операционные процессы «изнутри». 
Интересуюсь бизнес‑анализом и аналитикой данных: моделирую процессы в BPMN/UML, осваиваю SQL и Python для подтверждения гипотез данными.


## Навыки

- Процессы: BPMN (as‑is / to‑be), UML use case, базовые UML class‑диаграммы.
- Аналитика данных: базовый SQL (SELECT, JOIN, GROUP BY), базовый Python (Pandas в процессе).
- Инструменты: AnyLogic (имитационные модели), Miro, MS Project, GitHub.


## Проекты

### 1. [Внедрение терминала самообслуживания в компьютерном клубе](https://github.com/sa1ly-collap/project-selfservice-terminal)
Имитационная модель процесса пополнения игрового баланса и сравнение сценариев обслуживания (через администратора и через ТСО).

**Проделанная работа:** Описала процесс в BPMN, собрала статистику посещаемости, построила имитационную модель в AnyLogic и сравнила сценарии as-is/to-be.

**Инструменты:** BPMN, AnyLogic

### 2. [Разработка проекта внедрения информационной системы учета и анализа операционной деятельности компьютерного клуба](https://github.com/sa1ly-collap/project-club-operations)

Проектирование информационной системы, которая объединяет учёт смен, уборок, инцидентов и выручки в едином контуре и снижает долю ручных операций.

**Проделанная работа:** описала текущие процессы операционной деятельности в BPMN (as‑is), спроектировала целевые процессы с использованием единой системы (to‑be) и предложила целевую архитектуру ИС.
